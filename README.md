# Screen Aspect Ratio Mod

This mod allows you to **stretch or narrow the screen**, effectively changing the **aspect ratio**.

## Installation

The mod works on **Fabric 1.21.1**.  
Simply place the mod file into the `mods` folder and make sure **Fabric API** is installed.
